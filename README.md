# LearnPlaywright

A comprehensive learning repository for Playwright testing framework and JavaScript concepts.

│   ├── 03_verify_setup.js

## Chapters

### Chapter 1: Basics
Introduction to Playwright fundamentals and JavaScript concepts with step-by-step examples.

- **01_basics.js** - Basic Playwright setup and usage
- **02_JS_step_by_step.js** - JavaScript concepts explained step by step
- **03_verify_setup.js** - Verify your Playwright setup
- **04_hot_code.js** - Hot code reload examples

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/TejaB444/LearnPlaywright.git
cd LearnPlaywright

# Install dependencies
npm install

# Install Playwright browsers
npx playwright install
```

### Running Examples

```bash
# Run a specific script
node chapter_01_Basics/01_basics.js

# Verify setup
node chapter_01_Basics/03_verify_setup.js
```
